Current Projects

LXC-Rails: A Ruby on Rails application devoted to providing an interface for maintaining quasi-virtualized services on a Linux host supporting Application and OS Linux Containers via LXC (Linux Containers)--tightly integrates Linux CGroups for resource management.

CalMon: An application that provides a Server-Client infrastructure for monitoring back-end service startup and shutdown over time via a multi-language client interface, including BASH and NetCat. Implements the Google Calendar API for creating, modifying, and deleting ‘event’s in real-time.

Rye with DSL: The latest version of Rye, a tool for running commands on many servers (like Capistrano), includes a DSL I wrote in the hopes to emulate some of the configuration behaviors of the Python tool Fabric.
